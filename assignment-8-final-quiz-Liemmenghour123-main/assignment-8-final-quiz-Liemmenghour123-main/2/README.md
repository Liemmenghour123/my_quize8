# 문제 2. 환경변수 설정 스크립트 작성

`2/` 디렉터리에는 아래 실행 파일이 제공되어 있다.

```text
2/env-report
```

환경변수를 설정하는 쉘 스크립트를 아래 경로에 만든다.

```text
2/final-env.sh
```

설정해야 하는 환경변수와 값은 아래와 같다.

```text
FINAL_QUIZ_MODE=final
FINAL_REPORT_NAME=env-result
FINAL_SCORE_LIMIT=80
FINAL_DATA_FILE=data/scores.txt
```

환경설정 파일을 적용한 뒤 `2/env-report`를 실행한다.

`2/env-report`는 결과 파일을 자동으로 만든다.

필수 산출물은 아래와 같다.

```text
reports/env-result.txt
```

## Problem 2. Write an environment variable setup script

The `2/` directory contains the executable below.

```text
2/env-report
```

Create a shell script that sets environment variables at the path below.

```text
2/final-env.sh
```

Set the environment variables to the values below.

```text
FINAL_QUIZ_MODE=final
FINAL_REPORT_NAME=env-result
FINAL_SCORE_LIMIT=80
FINAL_DATA_FILE=data/scores.txt
```

After applying the environment configuration file, run `2/env-report`.

`2/env-report` creates the result file automatically.

The required output file is below.

```text
reports/env-result.txt
```
